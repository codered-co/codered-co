--- title: Bootcamp Android description: Bem vindo a nosso bootcamp de android. ---

Javeiro bronze - Introdução e primeiros passos (21 de setembro)

Na primeira oficina de Java você irá aprender os conceitos básicos da linguagem, tais como:
  • Tipos primitivos
  • Casting
  • Operadores Condicionais
  • Operadores Lógicos
  • Estruturas de Repetição
  • Introdução ao paradigma Orientado a Objetos

Javeiro prata - Avançando na linguagem (14 de setembro)

Na segunda oficina de Java você irá se aprofundar no paradigma de programação orientado a objetos com os tópicos:
  • Modificadores de visibilidade
  • Construtores
  • Métodos estáticos
  • Herança
  • Polimorfismo
  • Exceções
  • Refatoração

Bootcamp - Desenvolvendo aplicações móveis (28 de setembro)

No Bootcamp você irá se aprofundar no desenvolvimento para dispositivos móveis, onde por sua vez colocará em prática o conhecimento adquirido até o momento nas oficinas de java:
  • Introdução ao SO Android
  • Arquitetura do Android
  • Criação de telas
  • Fluxo de Navegação
  • Meu primeiro app
  • Listas Customizadas
